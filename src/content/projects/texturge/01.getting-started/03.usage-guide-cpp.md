# 使用指南：C++ 版本

> [!WARNING]
> 此文档为 C++ 使用指南的占位草稿，C++ API 尚未经过完整测试验证。当前推荐使用蓝图版本。

## 纯文本打字机效果

```cpp
// 创建 AnimatedTextBlock
UAnimatedTextBlock* TextBlock = NewObject<UAnimatedTextBlock>();
TextBlock->SetText(FText::FromString("Hello Texturge!"));
TextBlock->TextAnimationBlueprint = LoadObject<UTextAnimationBlueprint>(
    nullptr, TEXT("/Game/Animations/TABP_Typewriter"));
TextBlock->bAutoPlay = true;

// 绑定事件
TextBlock->OnCharacterRevealedBP.AddDynamic(this, &AMyActor::OnCharRevealed);
TextBlock->OnAnimationCompleteBP.AddDynamic(this, &AMyActor::OnAnimComplete);
```

## 多格式文本多动画混合

```cpp
// 加载 DataAsset
UTextAnimationDataAsset* DataAsset = LoadObject<UTextAnimationDataAsset>(
    nullptr, TEXT("/Game/Animations/DA_Dialog"));

// 创建 AnimatedRichTextBlock
UAnimatedRichTextBlock* RichBlock = NewObject<UAnimatedRichTextBlock>();
RichBlock->SetText(FText::FromString(
    "<typewriter>Hello adventurer!</> <bounce>Welcome</>"));
RichBlock->AnimationData = DataAsset;
RichBlock->bAutoPlay = true;

// 绑定事件
RichBlock->OnAnimationCompleteBP.AddDynamic(this, &AMyActor::OnDialogComplete);
```
